# Kyopro Encyclopedia of Absentmind (ア！ｗ 辞典)

<https://nebocco.github.io/absendminded-encyclopedia/>

## これなに

競プロで犯しがちなミスを集めた百科辞典を目指しているサイトです。

### 内容の方針

-   読者は AtCoder 黄色以上を想定しています。青色以下が黄色になるために読むものではなく、黄色や橙色が赤色以上を目指すために読むようなものを書いています。赤色以上の人が読んで面白いと感じられる内容を目指します。
-   具体例を列挙して察してもらうだけは避け、そのミスが何故発生したのかの原因を考えます。ただ気を付けるというだけではなく、どうすれば回避できたのかという対策の参考になる情報をまとめます。ノウハウ本です。
- このサイトは [Kyopro Encyclopedia of Algorithm](https://noshi91.github.io/algorithm-encyclopedia/) のパロディとして作成されました。
